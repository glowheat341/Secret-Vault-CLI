# 🔐 Secret Vault CLI

Безопасный локальный менеджер зашифрованных секретов с красивым интерфейсом командной строки. Храните пароли, API ключи и конфиденциальные данные в безопасности с военным шифрованием AES-256-GCM.

[English](README.md) | **Русский**

## ✨ Возможности

- 🔒 **Военное шифрование** - AES-256-GCM с деривацией ключей scrypt
- 💾 **Локальное хранение** - Ваши секреты никогда не покидают ваш компьютер
- 🎨 **Красивый CLI** - Интуитивный и цветной интерфейс
- 📋 **Интеграция с буфером обмена** - Копируйте секреты без отображения
- 🏷️ **Категории** - Организуйте секреты по категориям
- 🔍 **Поиск** - Быстро находите секреты по имени, категории или метаданным
- 📤 **Импорт/Экспорт** - Резервное копирование и восстановление хранилища
- 🔐 **Мастер-пароль** - Один пароль для разблокировки всех секретов
- ⚡ **Быстрый и легкий** - Построен на TypeScript и Node.js

## 📖 Использование

### Инициализация хранилища

При первом запуске вам будет предложено создать мастер-пароль:

```bash
vault add
```

### Добавить секрет

```bash
# Интерактивный режим
vault add

# С опциями
vault add -n "github-token" -v "ghp_xxxxxxxxxxxx" -c "api-keys"
```

### Получить секрет

```bash
# Интерактивный режим (выбор: скопировать или показать)
vault get github-token

# Скопировать в буфер обмена
vault get github-token --copy

# Показать в терминале
vault get github-token --show
```

### Список секретов

```bash
# Показать все секреты
vault list

# Фильтр по категории
vault list --category api-keys
```

### Поиск секретов

```bash
vault search github
```

### Удалить секрет

```bash
vault delete github-token
```

### Категории

```bash
# Список всех категорий с количеством секретов
vault categories
```

### Экспорт/Импорт

```bash
# Экспорт хранилища в зашифрованный файл
vault export -o backup.enc

# Импорт хранилища из зашифрованного файла
vault import -i backup.enc
```

### Заблокировать хранилище

```bash
vault lock
```

## 🔒 Безопасность

- **Шифрование AES-256-GCM** - Промышленный стандарт аутентифицированного шифрования
- **Деривация ключей Scrypt** - Функция, устойчивая к брутфорсу
- **Случайные соли и IV** - Каждое шифрование использует уникальные случайные значения
- **Теги аутентификации** - Обеспечивает целостность и подлинность данных
- **Только локальное хранение** - Без облака, без сетевых запросов
- **Мастер-пароль** - Никогда не сохраняется, используется только для деривации ключей

## 📁 Расположение хранилища

```

```

## 📋 Справка по командам

| Команда | Описание | Опции |
|---------|----------|-------|
| `vault add` | Добавить или обновить секрет | `-n, --name`, `-v, --value`, `-c, --category` |
| `vault get <name>` | Получить секрет | `-c, --copy`, `-s, --show` |
| `vault list` | Список всех секретов | `-c, --category` |
| `vault delete <name>` | Удалить секрет | - |
| `vault search <query>` | Поиск секретов | - |
| `vault categories` | Список всех категорий | - |
| `vault export` | Экспорт хранилища | `-o, --output` |
| `vault import` | Импорт хранилища | `-i, --input` |
| `vault lock` | Заблокировать хранилище | - |
